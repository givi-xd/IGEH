# Install
```bash
 pkg update && pkg upgrade
 pkg install python
 pkg install git  
 pip insttall requests bs4 tqdm colorama
 rm -rf IGEH
 git clone https://github.com/givi-xd/IGEH
 cd IGEH
 chmod +x givi32 givi64
```
# HOW TO RUN?
### FOR 32 BIT
```bash
 ./givi32
```
### FOR 64 BIT
```bash
 ./givi64
```
